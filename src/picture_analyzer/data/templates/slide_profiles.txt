=== SLIDE RESTORATION PROFILES (MANDATORY - ALWAYS ANALYZE) ===
19. **Suggested Profiles** (MANDATORY FOR EVERY IMAGE): Analyze if this appears to be a scanned slide, dia positive, or vintage photograph, and suggest the most suitable restoration profiles with confidence scores (0-100%).
    
    GEMMA4 NOTE: This is a MANDATORY SECTION. Even if the image looks well-preserved, you MUST return at least one profile suggestion.
    
    Available profiles: 
    - faded: Very faded slides with lost color/contrast and washed-out appearance
    - color_cast: Generic color casts (any unusual color tint not covered by other profiles)
    - red_cast: Red/magenta aging (typical of old Kodachrome slides, reddish tint)
    - yellow_cast: Yellow/warm aging (golden/sepia tone, typical of older films)
    - aged: Moderate aging with slight color cast and contrast loss
    - well_preserved: Minimal aging, good color and contrast retention
    
    CRITICAL FIELD NAMES: Use EXACTLY "profile" and "confidence" (lowercase, no underscore).
    
    RESPONSE FORMAT - ALWAYS return a valid JSON array with 2-3 suggestions:
    - For ANY image, ALWAYS include "well_preserved" as baseline
    - Confidence values MUST be integers 0-100 (no decimals)
    - NEVER return an empty array []
    
    Examples of valid responses:
    - Faded slide: [{{"profile": "faded", "confidence": 85}}, {{"profile": "well_preserved", "confidence": 50}}]
    - Red-cast slide: [{{"profile": "red_cast", "confidence": 80}}, {{"profile": "aged", "confidence": 60}}, {{"profile": "well_preserved", "confidence": 40}}]
    - Yellow/orange vintage photo (1970s-80s): [{{"profile": "yellow_cast", "confidence": 85}}, {{"profile": "color_cast", "confidence": 65}}, {{"profile": "well_preserved", "confidence": 35}}]
    - Well-preserved photo: [{{"profile": "well_preserved", "confidence": 90}}, {{"profile": "aged", "confidence": 45}}]
    - Unknown aging: [{{"profile": "aged", "confidence": 60}}, {{"profile": "well_preserved", "confidence": 70}}]
    
    DETECTION RULES:
    1. If filename contains "dia", "slide", "transparency", "positive" → DEFINITELY provide profiles
    2. If image appears to be from before 2000 (especially 1970s-1980s) → expect strong color casts (orange/yellow/red)
    3. If image has strong orange/yellow tint → use yellow_cast (high priority for 1970s-80s)
    4. If image has red/magenta tint → use red_cast
    5. If image has visible grain, dust, fading, or ANY color cast → match closest profile
    6. If unsure between profiles → always include "well_preserved" with high confidence and one other profile with 50-60% confidence
    6. ALWAYS provide at least 2-3 profile recommendations including well_preserved as baseline
    7. well_preserved should ALWAYS be included (even if confidence is lower) as it's a safe baseline option
    8. If uncertain, default to "aged" or "well_preserved" with appropriate confidence (never return empty array for real photographs)
    
    IMPORTANT: Even scanned slides with minimal issues should get "well_preserved" profile. NEVER return [] for actual photographs.
